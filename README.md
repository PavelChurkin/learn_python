# learn_python

Учебный курс по языку программирования Python с примерами кода.

## Описание

Этот курс охватывает базовую теорию Python и включает теоретический материал, подробные пояснения и рабочие примеры кода. Каждая строка кода сопровождается комментариями на русском языке.

## Структура курса

Курс состоит из 8 модулей, которые можно запускать независимо друг от друга:

| Модуль | Тема | Файл |
|--------|------|------|
| 1 | Введение в Python: синтаксис, переменные, типы данных | `modules/module_01_introduction.py` |
| 2 | Операторы и выражения | `modules/module_02_operators.py` |
| 3 | Условные конструкции (if/elif/else) | `modules/module_03_conditionals.py` |
| 4 | Циклы (for, while) | `modules/module_04_loops.py` |
| 5 | Функции | `modules/module_05_functions.py` |
| 6 | Структуры данных (списки, кортежи, словари, множества) | `modules/module_06_data_structures.py` |
| 7 | Объектно-ориентированное программирование | `modules/module_07_oop.py` |
| 8 | Советы по оптимизации производительности | `modules/module_08_optimization.py` |

## Как использовать

### Запуск отдельного модуля

```bash
python3 modules/module_01_introduction.py
```

### Запуск всех модулей последовательно

```bash
for i in modules/module_*.py; do python3 "$i"; done
```

## Содержание модулей

### Модуль 1: Введение в Python
- Функция print() для вывода данных
- Переменные и правила именования
- Типы данных: int, float, str, bool, None
- Преобразование типов
- Форматирование строк (f-строки, format(), %)
- Комментарии в коде

### Модуль 2: Операторы и выражения
- Арифметические операторы (+, -, *, /, //, %, **)
- Операторы сравнения (==, !=, <, >, <=, >=)
- Логические операторы (and, or, not)
- Операторы присваивания (=, +=, -=, и др.)
- Побитовые операторы
- Операторы принадлежности (in, not in) и идентичности (is, is not)
- Приоритет операторов

### Модуль 3: Условные конструкции
- Оператор if
- Оператор if-else
- Оператор if-elif-else
- Вложенные условия
- Тернарный оператор
- Оператор match-case (Python 3.10+)

### Модуль 4: Циклы
- Цикл for и функция range()
- Цикл while
- Операторы break, continue, pass
- Функции enumerate(), zip(), reversed(), sorted()
- Генераторы списков (list comprehensions)

### Модуль 5: Функции
- Определение и вызов функций
- Параметры и аргументы
- Значения по умолчанию
- *args и **kwargs
- Области видимости (global, nonlocal)
- Лямбда-функции
- Декораторы
- Рекурсия

### Модуль 6: Структуры данных
- Списки (list): создание, индексация, методы
- Кортежи (tuple): неизменяемость, распаковка
- Словари (dict): ключи и значения, методы
- Множества (set): уникальность, операции
- Сравнение структур данных

### Модуль 7: Объектно-ориентированное программирование
- Классы и объекты
- Атрибуты и методы
- Инкапсуляция и свойства (@property)
- Наследование и super()
- Множественное наследование
- Полиморфизм
- Специальные методы (__init__, __str__, __add__ и др.)
- Классовые и статические методы
- Абстрактные классы

### Модуль 8: Советы по оптимизации
- Выбор правильных структур данных
- Оптимизация циклов
- Использование встроенных функций
- Кэширование с @lru_cache
- Работа с генераторами
- Профилирование кода (cProfile, timeit)

## Требования

- Python 3.8 или выше
- Для модуля 3 (match-case): Python 3.10 или выше
